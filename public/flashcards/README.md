# Slike za kartice

Ovdje idu slike (skice i izvodi formula) koje se prikazuju uz odgovore na karticama.

- Format: `.png` (ili `.jpg` — samo promijeni naziv u `src/data/flashcards.ts`).
- Datoteke se u aplikaciji dohvaćaju s putanje `/flashcards/<naziv>`.
- Ako datoteka nedostaje, kartica prikazuje poruku „Slika nije dostupna: <naziv>”.
